## Usage


### Config

Create a config.js file in then root and add the following

```
module.exports = {
  MONGODB: "your_mongodb_uri",
  SECRET_KEY: "yoursecret",
};
```

### Install Dependencies (frontend & backend)

```
npm install
cd client
npm install
```

### Run

```
# Run frontend (:3000) & backend (:5000)
npm run dev

# Run backend only
npm run server
```
